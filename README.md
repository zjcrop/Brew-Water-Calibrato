## 最简部署

1. 解压本包。
2. 上传全部文件到仓库根目录并覆盖同名文件。
3. 将Pages Source设为GitHub Actions。
4. 查看两个Actions工作流。
5. 下载`Brew-Water-Calibrato-v2.6.1-debug`构件。

详细步骤见`WORKFLOW_CONFIG_v2.6.1.md`。
